# Claude Code Conversation History

This folder contains exported conversation history from Claude Code sessions used to develop this project.

## Files

- `conversation_*.jsonl` - JSONL format conversation logs from Claude Code sessions

## Format

Each line in the JSONL files contains a JSON object representing a message in the conversation, including:
- User messages
- Claude's responses
- Tool calls and results

## Privacy Notice

These files may contain sensitive information from development sessions. Review before committing to a public repository.
